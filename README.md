# Prueba Técnica – Banco

## Descripción
Aplicación backend desarrollada en Java con Spring Boot para la gestión de clientes, productos financieros y transacciones.

## Funcionalidades
- CRUD de clientes
- Creación y gestión de cuentas de ahorro y corriente
- Registro de transacciones (consignación, retiro y transferencias)
- Consulta de estados de cuenta

## Arquitectura
- Controller
- Service
- Repository
- Entity
- DTO
- Exception

## Tecnologías
- Java
- Spring Boot
- Maven
- JPA / Hibernate
- JUnit

## Ejecución
```bash
mvn spring-boot:run
